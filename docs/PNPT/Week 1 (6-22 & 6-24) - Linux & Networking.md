## Ping Sweep Script

```bash
#!/bin/bash 

if [ "$1" == "" ] 
then 
echo "You forgot an IP address!" 
echo "Syntax: ./ipsweep.sh 192.168.5" 
else 
for ip in `seq 1 254`; do ping -c 1 $1.$ip | grep "64 bytes" | cut -d " " -f 4 &
done
fi
```
